# StartProject
### Overview
Starter project for android application. Project includes frequently used libraries.
### Features
Kotlin DSL for build scripts. Codestyle settings (ktlint, detekt).
### Libraries and dependencies
* Coroutines
* Retrofit
* Room
* Navigation (with Safeargs)
* Hilt
* Coil
* Timber
* Spek
